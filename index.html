NAE Index - Open Source Project

About the Project

The NAE Index is an open source project designed to simulate a small-scale stock market. Users can invest in stocks using tokens, view historical price data, and manage their accounts. The project includes both a user client and an admin board for managing stocks and publishing news articles.

Features

  • Invest in and withdraw from stocks using tokens
  • View historical price data with interactive graphs
  • Create and manage user accounts
  • Admin board for creating new stocks and adding tokens
  • Publish news articles related to stock indexes

How to Contribute

We welcome contributions from the community! To contribute, follow these steps:

  1. Fork the repository on GitHub
  2. Clone your forked repository to your local machine
  3. Create a new branch for your feature or bug fix
  4. Make your changes and commit them with clear messages
  5. Push your changes to your forked repository
  6. Submit a pull request to the main repository

For more detailed instructions, please refer to our contributing guidelines.
